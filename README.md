# 余荫山房数字化导览

手机端优先的沉浸式 H5 导览页，用于风景园林课程期末展示。

## 项目主题

- 标题：余荫山房数字化导览
- 副标题：岭南园林空间、建筑与游览体验展示
- 展示对象：广州番禺余荫山房
- 技术栈：HTML、CSS、JavaScript

## 功能

- 接近一屏高度的沉浸式 section
- IntersectionObserver 淡入动画
- 横向滑动拍照推荐卡片
- 点击展开园林元素卡片
- 图片灯箱放大预览
- GitHub Pages 自动部署工作流

## 图片说明

- 实景图片来自课程调查报告中整理的余荫山房照片。
- 补充图片基于余荫山房公开景点资料和 Wikimedia Commons 中的余荫山房视觉参考生成，不直接搬运带水印或版权不清的第三方图片。
- 03、04、05 三个内容模块共展示 13 张不同图片，避免重复。

## 本地预览

直接打开 `index.html`，或在项目目录运行一个静态服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/
```
