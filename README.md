# 余荫山房数字化导览

手机端优先的沉浸式 H5 导览页，用于风景园林课程期末展示。

## 项目主题

- 标题：余荫山房数字化导览
- 副标题：岭南园林空间、建筑与游览体验展示
- 展示对象：广州番禺余荫山房
- 技术栈：HTML、CSS、JavaScript

## 功能

- 接近一屏高度的沉浸式 section
- IntersectionObserver 淡入动画
- 横向滑动拍照推荐卡片
- 点击展开园林元素卡片
- 图片灯箱放大预览
- GitHub Pages 自动部署工作流

## 本地预览

直接打开 `index.html`，或在项目目录运行一个静态服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/
```
